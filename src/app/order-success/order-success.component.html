O Aplikacji

Aplikacja została stworzona w celach edukacyjnych oraz z pasji do zagadnień związanych z dziedziną Front-End.

Back-End tej aplikacji został stworzony w Firebase

Wszystkie przedmioty w tym sklepie pochodzą z bazy danych.

Technologie użyte w Front-End to: SASS, HTML5, CSS3, Angular 2+ 4, Bootstrap 4, Angular Material, JavaScript ES5, ES6 oraz TypeScript.

Jest to aplikacja niekomercyjna. Wszystkie grafiki zostały pobrane z google, a inspiracją dla produktów były: Komputronik, Morele, X-kom, Ceneo
